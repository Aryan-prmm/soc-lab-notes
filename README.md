## Lab 2: SOC L1 Alert Triage (TryHackMe)

### Scenario

### Evidence Reviewed
Indicators, logs, and contextual information associated with the alerts.

### Analysis
Evaluated alert evidence to differentiate between true and false positives using context and impact.

### Decision
Alerts were either closed as false positives or escalated as true positives with justification.

### SOC Perspective
This lab reinforced the importance of accurate alert triage to reduce alert fatigue and ensure timely escalation of genuine threats.
